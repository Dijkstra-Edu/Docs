**Duration:** 175-350 Hours \
**Difficulty:** Medium \
**Languages & Technologies:** TypeScript \
**Repository:** [Dijkstra-Edu/Dijkstra-Blog](https://github.com/Dijkstra-Edu/Blog), [Dijkstra-Edu/Dijkstra-Web](https://github.com/Dijkstra-Edu/Dijkstra-Web)

This project involves improving the blog system that Dijkstra currently employs.

## Objectives

- Enable GitHub authentication, as well as cross management with the comments system
- Creation of the Add and Edit Blog System on Dijkstra Web
- Administration mechanism for Dijkstra HQ
- Addressing Miscelaneous Bugs

## Requirements & Relevant Skills

- Experience with web development (React, Vue, or similar)
- Understanding of content management systems
- Knowledge of frontend frameworks and libraries
- Familiarity with backend APIs and data management
- Understanding of markdown/rich text editing

## Deliverables

- Fully functional blog system
- Content management interface
- Integration with Dijkstra Web
- Integration with Dijkstra HQ
- Integration with Archivist
- Documentation
