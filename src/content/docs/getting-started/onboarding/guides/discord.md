## What is Discord?

Discord is a communication platform that allows text, voice, and video interaction.
We use it for:
- Announcements
- Discussions
- Study sessions
- Community bonding

---

## Step-by-Step Setup

### 1. Create a Discord account
- Visit https://discord.com/register
- Sign up using email or Google

![Alt text](/images/onboarding/discord-step1.png)

### 2. Join our Discord server
- A server is a community space
- Channels organize conversations by topic
- Use the invite link provided. <a href="https://discord.gg/e9daneY9dW" target="_blank">Dijkstra Discord Server</a>
- Read the rules channel

![Alt text](/images/onboarding/discord-step2.png)

### 3. Voice channels & muting
- Join voice channels when required
- Mute mic when not speaking

![Alt text](/images/onboarding/discord-step3.png)

### 4. Set your nickname & profile picture
- Use your real name or preferred name
- Add a clear profile picture

![Alt text](/images/onboarding/discord-step4.png)

### 5. Add friends
- Make friends and share learn and grow together.

![Alt text](/images/onboarding/discord-step5.png)
