## What is VS Code?

Visual Studio Code is a lightweight but powerful code editor.
It supports:
- Multiple programming languages
- Extensions
- Integrated terminal

---

## Step-by-Step Setup

### 1. Download and install VS Code
- https://code.visualstudio.com


![Alt text](/images/onboarding/vscode-step1.png) ![Alt text](/images/onboarding/vscode-step1a.png) ![Alt text](/images/onboarding/vscode-step1b.png)![Alt text](/images/onboarding/vscode-step1c.png)

### 2. Install essential extensions
- Prettier: A professional editor who fixes your document's fonts and margins.
- ESLint: A proofreader who checks if your arguments make sense and catch typos.
- Copilot: A smart assistant who helps you draft the next paragraph based on your notes. (optional)

![Alt text](/images/onboarding/vscode-step2.png)

### 3. Configure basic settings
- Enable autosave
- Set theme and font

![Alt text](/images/onboarding/vscode-step3.png)

### 4. Learn keyboard shortcuts
- Open terminal: `Ctrl + \``
- Command palette: `Ctrl + Shift + P`

![Alt text](/images/onboarding/vscode-step4.png)

### 5. Set up integrated terminal
- Integrations help communication between frontend and backend.
- Use terminal for <a className = "font-semibold underline" Link = "https://git-scm.com/install/windows">Git</a>, <a className = "font-semibold underline" Link = "https://nodejs.org/en">Node.js</a>, <a className = "font-semibold underline" Link = "https://www.python.org/downloads/">Python</a> commands, pertaining to your need.
- Git: Track code changes and safely undo mistakes.
- Node.js: Run JavaScript directly on your machine for backend tasks.
- Python: Power your projects with automation, data, and AI scripts.

![Alt text](/images/onboarding/vscode-step6.png)

### 6. Start a project!
- Create a new folder, and open it as editor here. You can add and edit files to it.
- Or you can also paste a repo link from github into the "Clone Repository" option to edit.

![Alt text](/images/onboarding/vscode-step5.png)
